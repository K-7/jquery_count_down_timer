Here you will find Jquery Plugin for Count Down Timer in HH:MM:SS format.

## How to use jQuery Count Down timer:

Download

Requirement :

<script src="jquery-1.9.1.js">
<script src="timer.js">

Usage:

HTML include statement
<div id="time1">02:00:10</div > //Give the time in HH:MM:SS format itself
Jquery
$(document).ready(function(){

var t1 = $("#time1").timer({
fontSize : 25,
fontWeight : 900,
fontColor : '#000',
startTime : "00:01:10", //Time in HH:MM:SS can be given even here
stopTime : "00:00:00", // Can pause the time at this 'stopTime'
onComplete : function () { alert("Hurray !!! have a blessed life.") }
});


//Stop the timer
t1.stop();


// Restart the timer
t1.restart();


// After stopping allows to continue
t1.proceed();

});
